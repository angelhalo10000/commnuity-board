# 電子自治会アプリ 要件定義

*作成日: 2026-03-31 / バージョン: 1.0*

---

## 1. プロジェクト概要

### 目的
50世帯・8班編成の自治会における情報共有をデジタル化する。
現状のLINE連絡網・紙の回覧板を置き換え、情報伝達の抜け漏れと会長の作業負担を削減する。

### 技術スタック
- バックエンド: Ruby on Rails（APIモード）
- フロントエンド: React + Vite
- アクセス: モバイル・PC両対応（レスポンシブ）

### 解決する課題

| 課題 | 解決アプローチ |
|------|--------------|
| 班長が連絡を流し損ねる | 会長がサイトに直接投稿、会員が直接参照 |
| 会員の意見が届くのに時間がかかる | （将来機能） |
| 紙の回覧板の準備・配布が手間 | デジタル回覧板（PDF/画像アップロード） |
| 班によって回覧板が回ってこない | サイトから常時閲覧可能 |

## 2. ロール・認証

| ロール | 認証方式 | 権限 |
|--------|---------|------|
| 会員 | 会員共通パスワード | 全員向けコンテンツの閲覧・検索 |
| 班長 | 班長共通パスワード（会員PWを包含） | 全員向け + 班長向けコンテンツの閲覧・検索 |
| 会長 | 個別ログイン（パスワードのみ） | 閲覧・検索 + 投稿・管理 |

### 認証フロー
- 未認証ユーザーはパスワード入力画面にリダイレクト
- 入力されたパスワードを下記の順で照合し、最初に一致したロールのセッションを付与
  1. 班長共通PWと一致 → 班長セッション（班長向け＋全員向けコンテンツ閲覧可）
  2. 会員共通PWと一致 → 会員セッション（全員向けコンテンツのみ閲覧可）
  3. いずれにも不一致 → アクセス拒否
- 会長は `/admin` にアクセスし、パスワードのみで管理機能にログイン

### パスワードバリデーション
- 会員共通PWと班長共通PWは異なる値でなければならない
- 会員共通PWの変更時：新しい値が現在の班長共通PWと一致する場合は更新を拒否
- 班長共通PWの変更時：新しい値が現在の会員共通PWと一致する場合は更新を拒否

## 3. 機能要件

### 3.1 公開側（共通パスワード認証後）

#### お知らせ
- トップページに最新お知らせ一覧を表示
- 配信日から30日以内のお知らせに「新着」バッジを表示（既読管理なし）
- お知らせ詳細を閲覧（タイトル・本文・添付ファイル）
- 日付検索（月・年での絞り込み）
- キーワードによる全文検索（タイトル・本文）
- 日付 + キーワードの組み合わせ検索

#### 回覧板
- トップページに最新回覧板一覧を表示
- 回覧板一覧・検索（キーワード、年月での絞り込み）
- 回覧板詳細を閲覧（PDF/画像のインラインビューア）

### 3.2 管理側（会長ログイン後）

#### 認証
- パスワードでログイン・ログアウト

#### お知らせ管理
- お知らせ作成（タイトル・本文・添付ファイル・配信対象：全員 or 班長のみ）
- お知らせ編集・削除
- 下書き保存

#### 回覧板管理
- 回覧板作成（タイトル・PDF/画像アップロード・配信対象：全員 or 班長のみ）
- 下書き保存・予約配信
- 回覧板削除

#### 設定
- 会員共通パスワード・班長共通パスワードの変更（両者は異なる値でなければならない）
- 管理者パスワードの変更
- 自治会名の変更

## 4. 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンシブ | スマートフォン・PC・タブレット対応 |
| ブラウザ | Safari・Chrome・Firefox 最新2バージョン |
| フォントサイズ | 最小14px以上（高齢者配慮） |
| 通信 | HTTPS必須 |
| パスワード | bcryptハッシュ化 |
| ファイルサイズ | 1ファイルあたり最大10MB |

## 5. 画面一覧

```
[共通パスワード認証]
  └── /enter            パスワード入力画面

[会員・班長（認証後）]
  ├── /                 トップページ（お知らせ一覧 + 回覧板一覧）
  ├── /notices          お知らせ一覧・検索
  ├── /notices/:id      お知らせ詳細
  ├── /circulars        回覧板一覧
  └── /circulars/:id    回覧板詳細

[会長管理画面]
  ├── /admin/login                        ログイン
  ├── /admin/password_reset               パスワードリセット（メール入力）
  ├── /admin/password_reset/sent          送信完了
  ├── /admin/password_reset/edit?token=…  新しいパスワード設定（メールリンク）
  ├── /admin/                   ダッシュボード
  ├── /admin/notices            お知らせ一覧
  ├── /admin/notices/new        お知らせ作成
  ├── /admin/notices/:id/edit   お知らせ編集
  ├── /admin/circulars          回覧板一覧
  ├── /admin/circulars/new      回覧板作成
  └── /admin/settings           設定（共通パスワード変更）
```

## 6. データモデル

ファイルストレージ: Cloudflare R2（Active Storage / S3互換）

```
organizations（自治会）
  ├── id                       # UUID7
  ├── name
  ├── member_password_digest   # 会員共通パスワード
  ├── leader_password_digest   # 班長共通パスワード
  ├── created_at
  └── updated_at

blocks（班）※将来機能用
  ├── id                       # UUID7
  ├── organization_id
  ├── name
  ├── sort_order
  ├── created_at
  └── updated_at

admin_users（会長）
  ├── id                       # UUID7
  ├── organization_id
  ├── password_digest
  ├── created_at
  └── updated_at

notices（お知らせ）
  ├── id                       # UUID7
  ├── organization_id
  ├── admin_user_id
  ├── title
  ├── body
  ├── target_type              # all / leaders
  ├── status                   # draft / scheduled / published / archived
  ├── scheduled_at             # 予約投稿日時（nullable）
  ├── published_at             # nullable
  ├── created_at
  └── updated_at
  # 添付ファイルは Active Storage で管理（notice_attachments テーブル不要）

circulars（回覧板）
  ├── id                       # UUID7
  ├── organization_id
  ├── admin_user_id
  ├── title
  ├── target_type              # all / leaders
  ├── status                   # draft / scheduled / published / archived
  ├── scheduled_at             # 予約配信日時（nullable）
  ├── published_at             # nullable
  ├── created_at
  └── updated_at
  # ファイル（PDF/画像）は Active Storage で管理（file_type は content_type から判定）
```

## 7. MVP スコープ

### MVP（初回リリース）
- [ ] 共通パスワード認証（会員・班長の2段階）
- [ ] 会長ログイン・ログアウト
- [ ] お知らせ作成・編集・削除・一覧・詳細（予約投稿含む）
- [ ] 回覧板作成・削除・一覧・詳細
- [ ] お知らせ検索（日付・月・年・キーワード全文検索）
- [ ] 設定（会員共通PW・班長共通PW変更）

### 将来機能（v2以降）
- 配信対象班の絞り込み表示
- コメント・意見投稿
- アンケート機能
- 施設予約・イベントカレンダー
- マルチテナント対応
