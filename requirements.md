# 電子自治会アプリ 要件定義

*作成日: 2026-03-31 / バージョン: 1.0*

---

## 1. プロジェクト概要

### 目的
50世帯・8班編成の自治会における情報共有をデジタル化する。
現状のLINE連絡網・紙の回覧板を置き換え、情報伝達の抜け漏れと会長の作業負担を削減する。

### 技術スタック
- バックエンド: Ruby on Rails（APIモード）
- フロントエンド: React + Vite
- アクセス: モバイル・PC両対応（レスポンシブ）

### 解決する課題

| 課題 | 解決アプローチ |
|------|--------------|
| 班長が連絡を流し損ねる | 会長がサイトに直接投稿、会員が直接参照 |
| 会員の意見が届くのに時間がかかる | （将来機能） |
| 紙の回覧板の準備・配布が手間 | デジタル回覧板（PDF/画像アップロード） |
| 班によって回覧板が回ってこない | サイトから常時閲覧可能 |

## 2. ロール・認証

| ロール | 認証方式 | 権限 |
|--------|---------|------|
| 会員 | 会員共通パスワード | 全員向けコンテンツの閲覧・検索 |
| 班長 | 班長共通パスワード（会員PWを包含） | 全員向け + 班長向けコンテンツの閲覧・検索 |
| 会長 | 個別ログイン（メール + パスワード） | 閲覧・検索 + 投稿・管理 |

### 認証フロー
- 未認証ユーザーはパスワード入力画面にリダイレクト
- 共通パスワードを入力するとセッションに記録され、以降スキップ
- 会長は `/admin` にアクセスし、個別ログインで管理機能を利用

---

## 3. 機能要件

### 3.1 公開側（共通パスワード認証後）

#### お知らせ
- トップページに最新お知らせ一覧を表示
- 配信日から30日以内のお知らせに「新着」バッジを表示（既読管理なし）
- お知らせ詳細を閲覧（タイトル・本文・添付ファイル）
- 日付検索（月・年での絞り込み）
- キーワードによる全文検索（タイトル・本文）
- 日付 + キーワードの組み合わせ検索

#### 回覧板
- トップページに最新回覧板一覧を表示
- 回覧板詳細を閲覧（PDF/画像のインラインビューア）

### 3.2 管理側（会長ログイン後）

#### 認証
- メールアドレス・パスワードでログイン・ログアウト
- パスワードリセット（メール送信）

#### お知らせ管理
- お知らせ作成（タイトル・本文・添付ファイル・配信対象班）
- お知らせ編集・削除
- 下書き保存

#### 回覧板管理
- 回覧板作成（タイトル・PDF/画像アップロード・配信対象班）
- 回覧板削除

#### 設定
- 共通パスワードの変更

## 4. 非機能要件

| 項目 | 要件 |
|------|------|
| レスポンシブ | スマートフォン・PC・タブレット対応 |
| ブラウザ | Safari・Chrome・Firefox 最新2バージョン |
| フォントサイズ | 最小14px以上（高齢者配慮） |
| 通信 | HTTPS必須 |
| パスワード | bcryptハッシュ化 |
| ファイルサイズ | 1ファイルあたり最大10MB |

## 5. 画面一覧

```
[共通パスワード認証]
  └── /enter            パスワード入力画面

[会員・班長（認証後）]
  ├── /                 トップページ（お知らせ一覧 + 回覧板一覧）
  ├── /notices          お知らせ一覧・検索
  ├── /notices/:id      お知らせ詳細
  ├── /circulars        回覧板一覧
  └── /circulars/:id    回覧板詳細

[会長管理画面]
  ├── /admin/login              ログイン
  ├── /admin/                   ダッシュボード
  ├── /admin/notices            お知らせ一覧
  ├── /admin/notices/new        お知らせ作成
  ├── /admin/notices/:id/edit   お知らせ編集
  ├── /admin/circulars          回覧板一覧
  ├── /admin/circulars/new      回覧板作成
  └── /admin/settings           設定（共通パスワード変更）
```

## 6. データモデル

```
organizations（自治会）
  ├── id
  ├── name
  └── member_password_digest   # 共通パスワード

blocks（班）
  ├── id
  ├── organization_id
  ├── name
  └── sort_order

admin_users（会長）
  ├── id
  ├── organization_id
  ├── email
  └── password_digest

notices（お知らせ）
  ├── id
  ├── organization_id
  ├── admin_user_id
  ├── title
  ├── body
  ├── target_type              # all / blocks
  ├── status                   # draft / scheduled / published / archived
  ├── scheduled_at             # 予約投稿日時（nullable）
  ├── created_at               # 作成日時
  └── updated_at               # 更新日時（編集時に自動更新）
  └── published_at

notice_targets（配信対象班）
  ├── notice_id
  └── block_id

notice_attachments（添付ファイル）
  ├── id
  ├── notice_id
  └── file_url

circulars（回覧板）
  ├── id
  ├── organization_id
  ├── admin_user_id
  ├── title
  ├── file_url
  ├── file_type                # pdf / image
  ├── target_type              # all / blocks
  └── status                   # published / closed

circular_targets（配信対象班）
  ├── circular_id
  └── block_id
```

| 用語 | 説明 |
|------|------|
| 班 | 自治会を構成する小単位。8班編成 |
